
**Before doing anything type 'npm install'** from the command line. This will grab all the dependencies we need to run our react app. It might take a minute or so to run. 

Now run 'npm run start' to boot up the app. You can now navigate to localhost:3000 in the browser and you should see the app's main page. We'll implement the method necesssary to make the "Send Ether" button functional and also the code that logs events in the browser window.
